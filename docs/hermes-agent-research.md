# Hermes Agent Research

## Summary
Hermes Agent: autonomous development agent. Grows skills.

## Requirements
- OS: Linux/macOS/WSL2/Android.
- Infrastructure: Local/Cloud/Serverless.
- Backend: local, Docker, SSH, Daytona, Singularity, Modal.

## Installation
```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
hermes setup
```

## Interactive Use
- `hermes`: Start chat.
- `hermes setup`: Config providers/models.
- `SOUL.md`: Personality context.
- Skills: Learns from experience.
